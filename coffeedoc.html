Documentation functions

These functions extract relevant documentation info from AST nodes as returned by the coffeescript parser.

Functions

exports.documentModule(script)

Given a module's source code, returns module information in the form:

{
    "docstring": "Module docstring",
    "classes": [class1, class1...],
    "functions": [func1, func2...]
}

getNodes(script)

Generates the AST from coffeescript source code. Adds a 'type' attribute to each node containing the name of the node's constructor, and returns the expressions array

getClasses(nodes)

Returns all top-level class nodes from the AST as returned by getNodeTypes

getFunctions(nodes)

Returns all top-level named function nodes from the AST as returned by getNodeTypes

documentClass(cls)

Evaluates a class object as returned by the coffeescript parser, returning an object of the form:

{
    "name": "MyClass",
    "docstring": "First comment following the class definition"
    "parent": "MySuperClass",
    "methods": [method1, method2...]
}

documentFunction(func)

Evaluates a function object as returned by the coffeescript parser, returning an object of the form:

{
    "name": "myFunc",
    "docstring": "First comment following the function definition",
    "params": ["param1", "param2"...]
}

removeLeadingWhitespace(str)

Given a string, returns it with leading whitespace removed but with indentation preserved
